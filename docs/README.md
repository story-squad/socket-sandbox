# socket-sandbox

.env
```
PORT=8080
IP=127.0.0.1
```

clone the repo
```
git clone https://github.com/story-squad/socket-sandbox.git
cd socket-sandbox
```
install server libs
```
npm install
```
install client libs
```
cd client
yarn install
cd ..
```
build
```
npm run build
```
start
```
npm start
```
